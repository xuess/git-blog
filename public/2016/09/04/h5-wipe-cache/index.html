html5端设置 meta 标签：

设置上面的只是紧紧可以保证html文件每次从服务器中获取，不从缓存文件中拿，而对于外链CSS JS 图片 等文件仍旧是从缓存中获取的；

设置 css JS文件不从缓存中读取:

通过添加版本号的和随机数的方法，保证每次加载JS CSS连接都是最新的，通常的做法是添加一个版本号，在每次更新了JS CSS时给版本号+1，保证没有更新时采用缓存文件。

不推荐加随机数的方法，每次都是最新的，没办法利用缓存提高性能。
